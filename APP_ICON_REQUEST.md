# 앱 아이콘 제작 요청

## 앱 개요
**앱 이름**: 캐주얼 게임 모음 (Casual Game Collection)
**플랫폼**: iOS (iPhone/iPad)
**앱 타입**: 게임 모음 앱

## 앱 설명
간단하고 재미있는 캐주얼 게임들을 모아놓은 iOS 앱입니다. 사용자가 하나의 앱에서 여러 클래식 게임을 즐길 수 있습니다.

## 포함된 게임들
1. **테트리스** - 블록을 쌓아 라인을 완성하는 퍼즐 게임 (파란색 테마)
2. **오목** - 5개 돌을 연속으로 놓는 전략 게임 (갈색/나무색 테마)
3. **스도쿠** - 숫자 퍼즐 게임 (초록색 테마)
4. **스네이크** - 뱀을 조작하여 음식을 먹는 게임 (주황색 테마)

## 아이콘 디자인 요구사항

### 스타일
- **모던하고 미니멀한 디자인**
- **플랫 아이콘 스타일** (iOS 앱 아이콘 가이드라인 준수)
- **캐주얼하고 친근한 느낌**
- **게임의 재미와 즐거움을 표현**

### 컨셉 제안
다음 중 하나 또는 조합:
1. **게임 모음 컨셉**: 여러 게임 요소를 조합 (예: 테트리스 블록, 오목 돌, 스도쿠 숫자, 스네이크 등)
2. **게임 패드 컨셉**: 게임 컨트롤러나 게임 패드 모양
3. **퍼즐 조각 컨셉**: 여러 게임을 나타내는 퍼즐 조각들이 모인 형태
4. **게임 보드 컨셉**: 다양한 게임 보드가 겹쳐진 형태

### 색상
- **주 색상**: 보라색, 파란색, 주황색, 초록색 등 밝고 생동감 있는 색상
- **그라데이션 사용 가능** (iOS 스타일에 맞게)
- **대비가 뚜렷하여 작은 크기에서도 잘 보이도록**

### 디자인 요소
- 게임의 다양성을 표현할 수 있는 요소 포함
- 너무 복잡하지 않게 (아이콘 크기에서도 인식 가능하도록)
- iOS Human Interface Guidelines 준수

### 기술적 요구사항
- **크기**: 1024x1024 픽셀 (PNG 형식, 투명 배경 없음)
- **모서리**: iOS가 자동으로 둥글게 처리하므로 직각 사각형
- **배경**: 단색 또는 그라데이션 (투명 배경 불가)
- **해상도**: 고해상도 (Retina 디스플레이 대응)

### 참고사항
- 앱 내부에서 사용하는 색상: 파란색, 갈색, 초록색, 주황색
- 앱의 메인 화면 배경은 보라색-파란색 그라데이션 사용
- "🎮" 이모지가 앱 메인 화면에 사용됨

## 추가 요청사항
- 아이콘만 제공해주시면 됩니다 (다양한 크기 변형은 필요 없음)
- 아이콘 디자인에 대한 간단한 설명도 함께 제공해주시면 감사하겠습니다
